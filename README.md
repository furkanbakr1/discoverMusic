# MusicHub

Modern ve kullanıcı dostu bir müzik streaming platformu.

## Özellikler

- Responsive tasarım
- Müzik çalar kontrolü
- Arama fonksiyonu
- Kullanıcı girişi/kaydı
- Öne çıkan albümler ve öneriler
- Trend şarkılar listesi

## Teknolojiler

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
- JavaScript

## Kurulum

1. Repoyu klonlayın
2. `index.html` dosyasını bir tarayıcıda açın
